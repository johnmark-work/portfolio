# Portfolio Website

This is my personal portfolio website built using **Next.js**, **TypeScript**, and **Tailwind CSS**. It showcases my projects, skills, certificates, and provides a way to contact me.

## Features

- **Homepage** – Introduction and brief overview of who I am.
- **Projects** – My completed projects with descriptions and tools.
- **Tools** – Highlights the technologies and tools I use for development.
- **Certificates** – Showcases my professional and educational certificates.
- **Contacts** – Provides ways to reach me via email or social media links.
